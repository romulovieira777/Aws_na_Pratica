# Seção 02 — Introdução a AWS

Documentação da seção de introdução à AWS do curso **AWS na Prática**.

## Português (Brasil)

Esta seção apresenta os primeiros passos na AWS, o modelo de responsabilidade compartilhada, MFA, AWS CLI e links de apoio para consulta.

### Resumo dos arquivos

- `Slides/2.1_CriandoContaAWS.pdf`: primeiros passos na AWS, Free Tier, acesso ao console e links oficiais de apoio.
- `Slides/2.1_SharedResponsability.pdf`: modelo de responsabilidade compartilhada, responsabilidades do cliente e exemplos de uso.
- `Slides/2.2_MFA.pdf`: autenticação multifator na AWS, com exemplos de MFA virtual, U2F e hardware.
- `Slides/2.3_AWSCli.pdf`: visão geral da AWS CLI, versões disponíveis e instruções de instalação.
- `Documentos/Links.txt`: lista de links úteis para documentação, webinars, tutoriais, Free Tier, IAM e infraestrutura global.
